Your boss using Windows, and you using GNU/Linux, how to help her open Winamp from your desk?

December, 11 2013 - 10 minutes read



To answer this question, it was easy. Long title, but short way to solve it. The way called "Remote". It's control other operating system eventhough it is same Operating Systems or different. This activity similar with like remoting your television. Ok how ? Windows can be remoted by using Remote Desktop Protocol (RDP). This protocols is design to communicate other OS with Microsoft Windows Machine. To use remote with this protocol on GNU/Linux, use tiny apps but awesome called "rdekstop". But how to get this stuff: #apt-get install rdesktop Afer it finished, use with command : #rdesktop ip.computer.to.remoted i.e, we want remote computer with IP 192.168.2.2, then command : #rdesktop 192.168.2.2 If you want to display fullscreen, just add command : #rdesktop -A 192.168.2.2 and..tarra..., you can acces Microsoft Windows Machine from GNU/linux PS: More about RDP : http://support.microsoft.com/kb/186607 More about rdekstop : http://www.rdesktop.org/ This Screenshot from mine:
